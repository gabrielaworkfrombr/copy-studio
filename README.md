# Copy Studio

Ferramenta de produção de copy para redes sociais com fluxo de aprovação de pautas.

## Como usar

1. Acesse o app pelo Vercel (ou localhost)
2. Vá em **Configurações** e cole sua chave de API (Anthropic ou Gemini)
3. Vá em **Clientes** e importe um JSON de ICP ou cadastre manualmente
4. Vá em **Produção**, escolha o cliente e inicie o fluxo

## Fluxo de produção

1. Escolhe o modo: **Calendário semanal** ou **Copy avulsa**
2. Revisa e aprova as pautas geradas
3. Gera as copies das pautas aprovadas

## Deploy no Vercel

1. Suba este repositório no GitHub
2. Acesse [vercel.com](https://vercel.com) e importe o repositório
3. Clique em Deploy — sem configuração adicional necessária

## Stack

HTML + CSS + JS puro. Zero dependências. Zero build step.
